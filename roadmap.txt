我们的纪念册 (Our Memory) 产品演进蓝图 (2025 Comprehensive Edition)
=====================================================

核心战略: 
1. 流量 (Acquisition): 情头/AI 绘图 + 社交裂变
2. 留存 (Retention): 宠物情感羁绊 + 消息触达 + 互动内容
3. 变现 (Monetization): 激励视频 + 权益体系 + AI 增值服务


├── 🟢 阶段一：冷启动与流量突围 (v1.3.0) [当前版本·已上线]
│   ├── 🎁 流量钩子: 圣诞/元旦情头库、AI 动漫合照 (免费额度+裂变)
│   ├── 🐶 宠物 MVP: 基础喂食、旅行、明信片掉落
│   └── 📈 运营动作: 小红书/咸鱼铺量，VIP 口令 裂变
|
├── 🟢 阶段一 (续)：商业化与稳定性补丁 (v1.3.1) [目标: 快速变现 & 体验止血]
│   ├── 💰 广告变现接入 (Monetization):
│   │   ├── 核心逻辑: 接入微信激励视频广告组件 (Rewarded Video)
│   │   ├── 场景 A (高频): 宠物粮票补给
│   │   │   └── 触发条件: 喂食时 `food` < 10 或库存耗尽
│   │   │   └── 奖励: 观看完发放 3 份“豪华猫粮” (通过云函数 `add_item` 发放)
│   │   └── 场景 B (刚需): AI 胶卷/加速
│   │   │   └── 触发条件: 今日免费额度(1次)用完 或 勾选“优先极速通道”
│   │   │   └── 奖励: 单次 AI 生成权 + 去除水印
│   │
│   ├── 🐛 线上问题修复 (Bug Fixes):
│   │   ├── AI 生成超时优化:
│   │   │   ├── 后端: 调整 `process_anime` 云函数超时时间至 60s
│   │   │   └── 前端: 优化 Loading 态，增加“预计排队 15s，请勿退出”的进度提示，防止用户以为卡死
│   │   ├── 广告预加载 (Pre-load):
│   │   │   └── 逻辑: 在页面 `onLoad` 时静默加载广告流，避免用户点击按钮时转圈无响应
│   │   └── 异常兜底:
│   │       └── 修复部分机型保存图片到相册时的 `authorize` 权限死循环问题
│
├── 🟡 阶段二：品牌重塑与留存闭环 (v1.4.0) [春节前冲刺]
│   ├── 🔔 消息订阅系统 (Retention Core):
│   │   ├── 场景A (核心): 崽崽旅行归来通知 (Trigger: 出发时授权)
│   │   ├── 场景B (社交): 伴侣互动提醒 (Trigger: 每日任务/喂食时授权)
│   │   └── 场景C (习惯): 每日打卡/早安提醒
│   ├── 🧠 核心概念重构 (Brand Soul):
│   │   ├── 逻辑缝合: 宠物旅行明信片 -> 自动同步至“时光长廊”相册
│   │   └── 情感包装: 文案全面“拟人化” (喂食->投入爱意)
│   └── 📝 轻量级互动 (Time Killer):
│       └── 恋爱 100 件小事 (Bucket List):
│           ├── 预置清单: 内置 50-100 个经典情侣任务 (如: 一起看日出)
│           ├── 证据上传: 完成任务需上传照片解锁“成就勋章”
│           └── 社交分享: 生成精美的“打卡证书”海报
│
├── 🟠 阶段三：深度互动与内容生态 (v1.5.0) [目标: 提升时长 & 话题度]
│   ├── 🎭 AI 平行时空 (Love Simulator) [NEW]:
│   │   ├── 核心定义: 基于 LLM (大模型) 的情侣文字互动剧本 (非游戏类目)
│   │   ├── 玩法机制: 
│   │   │   ├── 剧本库: "古代穿越"、"末日求生"、"校园重逢"
│   │   │   ├── 双人决策: 两人针对剧情节点投票 -> 决定故事走向 -> 触发分歧或共鸣
│   │   │   └── 结局卡片: 生成“人生结局”海报 (如: 神仙眷侣/相忘江湖)
│   │   ├── 技术栈: 接入腾讯混元或 DeepSeek API + 文本安全校验
│   │   └── 变现点: 免费玩第一章，解锁“隐藏结局”或“特殊剧本”需看广告
│   ├── 🤝 互动升级:
│   │   ├── 串门机制: 去伴侣/好友家“偷吃”或“合影”
│   │   └── 收集图鉴: “旅行特产”点亮系统 (满足收集癖)
│   └── 💰 商业化变现 (流量主):
